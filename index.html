    Estrutura básica de um HTML

    (Voltar)

    Um documento HTML começa com a tag HTMLe termina com a marca /HTML. Ele também contém um cabeçalho que descreve o título da página e um corpo no qual se encontra o conteúdo da página. O cabeçalho é delimitado pelas tags HEAD e /HEAD. Já o corpo é delimitado pelas tags BODY e /BODY.

    Tags de texto

    (Voltar)

    Em HTML usamos algunas tags para inserção e formatação de textos, essas tags estão relacionadas a titulos, subtitulos, citações entre tantos outros itens para formular um texo apropriado para cada necessidade. São algunas delas:

    • H1 - Título, (H2 até H6 são para subtítulos)
    • strong - Define texto em Negrito (texto forte)
    • i - Define texto em itálico
    • u - Define texto em Sublinhado
    • sup - Define o texto em Superescrito
    • p - Define a inserção de parágrafos no texto

    Essas são apenas algumas das tags que vimos durante o curso em HTML, para mais conteúdos sobre tags e como fuincionam: Blog XPE Educação
